# **SmartOrders API**
## **Adding Points**
---
</div>

This command allows you to add points through the API.


### **Parameters**

| Name | Type | Description |
| ----------- | ----------- | ------- |
| User ID | Integer | The ID of the user. Can be gotten from `Player.UserId`.
| Points | Integer | The amount of points to add to the user.
